# Steam Sale Notifier

![Google Play Ready](https://play.google.com/static/client/images/435140425-play_logo.png)

## Description

Create a list of games from Steam and you'll be notified when Steam puts them on sale!

## Requirements

Steam Sale Notifier is designed to work with a minimum API level of 8. This means that the device needs to be running at least Android 2.2. However, the application was designed on a device running Android 4.1.
